5 wpi personality types

Realist

+ 타인의 인정을 통해 존재감 획득(공감이나 자기성찰 낮음)
+ 타인 시선 의식 → 너무 튀지 않게 남과 비슷하게 행동
+ 착한 사람 콤플렉스(배려, 순응, 모범)
+ 소속감을 통한 안정
(한국 사회 대부분의 조직인의 기본적인 정서와 성격)
+ 인생목표: 가정의 화복, 행복, 건강
+ 기존 틀에 충실. 큰 변화 거부, 새로운 시도 회피.
+ 카멜레온적인 상황 적응.
+ 부화뇌동으로 비춰질 가능성.

Romantist

+ 타인과의 교감을 통해 존재감 획득
+ 민감하고 불안정한 여성적이고 예술적인 감성. 멜랑콜리.
+ 미성숙하고 세상 경험이 적은 사춘기 소녀의 세상에 대한
두려움 기대, 걱정.
+ 완벽추구 (강한 자기확신)
+ 외적인 모습: 아름다움, 경제적 부유함 추구.
+ 일: 완결추구(결과지향적)

Humanist

+ 사람들과의 좋은 관계를 통해서 존재감 획득
+ 사교적 = 타인에 대한 관심 & 표현력(순발력)
+ 오지랖: 넓은 인간관계 네트워킹 형성
+ 마음 맞는 사람과 함께 일할 때 성과가 좋음.(동기가 중요)
+ 외향적, 긍정적: 복잡미묘한 감정파악에 서툶.
+ 부정적인 경험을 잘 잊음.
+ 카리스마가 있음. 권위적.
+ 좋은 인상을 얻기 위한 행동: 이벤트. 뒤처리.

Idealist

+ 세상에 대한 이해를 통해 자유를 느끼고 존재감 획득
+ 이상주의적. 창의적.
+ 에고이스트 성향. 고집.
+ 전문성. 탁월성 중시. 자뻑.
+ 독립적. 자유. 관행이나 의무 방어는 거부. 조직 관리에는 취약.
+ 정체성의 계속적인 확장.
+ 세상에 대해 자신이 이해한 바를 공유하고자 함. 화이부동.

Agent

+ 일을 통한 성취감에서 존재감 획득
+ 업무가 인간관계에 우선.
+ 결과지향적.
+ 유능함: 일에 대한 본인 스타일과 신뢰가 강함.
+ 전체 업무가 본인 틀과 맞을 때 추진력을 보임.
+ 반대의 경우 독선적, 단정적.
+ 일이든 취미든 강박적인 오타쿠 성향.
